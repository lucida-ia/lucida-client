# Identidade Visual — Lucida

Você é um especialista na identidade visual da Lucida. Ao criar qualquer elemento
visual ou de interface, siga rigorosamente estas diretrizes. Consulte os arquivos
em `references/` para detalhes completos quando necessário.

## Produtos da marca

| Produto              | Tom de cor | Quando usar       |
| -------------------- | ---------- | ----------------- |
| Lucida / Lucida Exam | Azul       | Produto principal |

→ Detalhes completos de cores: `references/colors.md`

## Tipografia — regras essenciais

**Principal:** Poppins — usar em tudo.
**Destaque:** Instrument Serif Italic — apenas para palavras de ênfase dentro de títulos.

Hierarquia:

- Título: Poppins Regular, 70pt, entrelinha 70
- Subtítulo: Poppins Regular, 40pt, entrelinha 40
- Corpo: Poppins Regular, 20pt, entrelinha 20
- Detalhe: Poppins Light, 18pt, entrelinha 20

**NUNCA:**

- Usar fontes fora da marca
- Esticar ou distorcer texto
- Usar outline em texto
- Justificar texto
- Kern ou entrelinha muito apertados

→ Detalhes de hierarquia e CSS: `references/typography.md`

## Logotipo — regras de uso

- Área de proteção mínima: equivalente à largura da letra "c" do logotipo em todos os lados
- Tamanho mínimo digital: **160px** (largura) para logotipo completo / **40px** para símbolo
- Posicionamento permitido: canto superior esquerdo, superior direito, centralizado,
  inferior esquerdo, inferior direito
- Cores permitidas: versão colorida (azul+preto), preto ou branco

**NUNCA:**

- Rotacionar o logotipo
- Mudar proporções
- Adicionar outline
- Mudar as cores fora do padrão
- Adicionar novos elementos ao logotipo

## Elementos gráficos

O símbolo da Lucida (o personagem "Lulu") pode ser usado em diferentes posições
e com "braços e ferramentas" para contextos específicos. Novas variações devem
seguir o padrão: apenas traços na cor preta, sem preenchimento, com pontas arredondadas.

## Checklist antes de entregar qualquer elemento visual

- [ ] Paleta de cores correta para o produto (Lucida/Exam = azul, Analytics = roxo, Learning = azul claro)
- [ ] Tipografia: apenas Poppins + Instrument Serif (só em destaques)
- [ ] Logotipo respeitando área de proteção e tamanho mínimo
- [ ] Texto não justificado, sem outline, sem distorção
- [ ] Cores secundárias usadas apenas como apoio/contraste
